# 数据可视化工具

一个功能丰富、易于使用的数据可视化工具，支持上传 Excel 和 CSV 文件，生成多种类型的交互式图表。

## 功能特性

### 文件支持
- Excel 文件 (.xlsx, .xls)
- CSV 文件 (.csv)
- 支持拖拽上传和点击选择
- 自动识别文件编码

### 数据统计摘要
- 自动统计数据行数和列数
- 识别数值列和文本列
- 显示常用统计指标（平均值等）

### 数据操作功能
- **数据筛选**: 支持全文搜索过滤数据
- **数据排序**: 按任意列升序或降序排列
- **数据聚合**: 
  - 分组统计
  - 支持求和、平均值、计数、最大值、最小值等聚合函数
  - 一键应用聚合结果

### 图表类型
- **柱状图** - 适合比较不同类别的数据
- **折线图** - 适合展示数据趋势变化
- **饼图** - 适合展示数据占比
- **环形图** - 饼图的变体，中心空心
- **雷达图** - 适合多维数据对比
- **极地图** - 极坐标下的面积图
- **散点图** - 适合展示两个变量的相关性
- **面积图** - 强调数量变化的图表

### 图表配置
- 可视化图表类型选择器
- 灵活选择 X 轴数据列
- **多系列数据支持**: 可同时显示多个 Y 轴数据系列
- 6 种配色方案：默认、柔和色、鲜艳色、冷色调、暖色调、单色系
- 4 种动画效果：默认、快速、慢速、无动画
- 5 种图例位置：顶部、底部、左侧、右侧、隐藏

### 主题切换
- 浅色/深色主题一键切换
- 自动保存用户偏好

### 导出功能
- 下载图表为 PNG 图片
- 下载图表为 JPG 图片
- 导出数据为 Excel 文件

## 使用方法

1. **上传文件**
   - 将 Excel 或 CSV 文件拖拽到上传区域
   - 或点击上传区域选择文件

2. **查看统计**
   - 上传后自动显示数据统计摘要
   - 查看数据行数、列数、列类型等

3. **数据操作（可选）**
   - 使用搜索框筛选数据
   - 选择排序列和排序方式
   - 点击列标题快速排序
   - 使用聚合功能分组统计数据

4. **配置图表**
   - 点击图标选择图表类型
   - 选择 X 轴数据列
   - 勾选一个或多个 Y 轴数据系列
   - 选择配色方案、动画效果和图例位置
   - 输入图表标题（可选）

5. **生成图表**
   - 点击"生成图表"按钮
   - 查看交互式图表

6. **导出结果**
   - 下载图表为 PNG 或 JPG 格式
   - 导出处理后的数据为 Excel 文件

## 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **图表库**: Chart.js 3.x
- **表格解析**: SheetJS (xlsx.js)
- **图标**: Font Awesome 6.x
- **样式特性**: CSS Variables, Flexbox, Grid, 动画

## 浏览器兼容性

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 本地运行

```bash
# 进入项目目录
cd data-visualizer

# 使用 Python 启动本地服务器
python3 -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 注意事项

1. 数据文件应包含表头行
2. 确保数据列格式正确，特别是数值列
3. 大数据集（超过1000行）可能需要更长的处理时间
4. 数据预览默认显示前100行
5. 建议使用现代浏览器以获得最佳体验
